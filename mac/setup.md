# set up mac

## システム環境設定
### トラックパッド
- 軌跡の速さ→最速
-  タップでクリック
- アプリExpose → 4本指で下にスワイプ
### キーボード
- リピート速度→最速
- リピート入力認識までの時間→最速
- キーボードショートカットの設定
	- ファンクションキー
		- F1、F2などのキーを標準のファンクションキーとして使用→ON
- 入力ソース
  - すべての入力ソース
	  - スマート引用符とスマートダッシュ→オフ
  - 日本語
	  -  ¥キーで入力する文字→バックスラッシュ
 - ユーザ辞書
